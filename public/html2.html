Basic HTML Chapter 2

Learning more about HTML starting with Doctype

Posted by VJ on June 2, 2015

Last chapter we learned what is html, tags and attribute, this chapter we will further learn about basics of html.

Text Editor

For creating a .html file you need a editor , there are some editor which by default comes with the OS.We will discuss couple of text editors which you can use to create or edit the HTML files.

One of the most common Text Editor you can find is Notepad in Windows OS.Notepad does not have any features related to the html but it is more than enough for the basic html craetion and saving.

The second text editor is Atom ,this is a free editor for MAC and Windows .This comes with a lot of features which makes the coding very easy and readable

If you can download and install software Atom will be a better option else you can use your default editor present in the OS

HTML Skeleton

A HTML file should have ceratin tags always in order for it to qualify,consider this as a skeleton of a human body.Like a human skeleton have different parts which forms the basic structure HTML also have parts which are required in all the html files

There are four necessary parts to all the html files

  • Main declaration
    • DOCTYPE
  • Three Main Tags
    • html
    • head
    • body

These tags you would have seen in the chapter one coding example, we will go through details about each of these tags individually.

Doctype

Below is the tag representation for Doctype

<!DOCTYPE html>

Doctype tells the browser what kind of file is loaded ,the above tag tells the browser that we are loading a HTML 5 file.Since HTML 5 is the latest specification I have used this.

Notice Doctype does not have a closing tag.It should always be at the begining of the file.It can only be used once in a given Page and it is the only tag which needs to be in upper case.

HTML tag

An attribute is an additional value that can configure a tag or adjust the behaviour of the tag.Below is an example for the attribute

<p id="welcome" class="text-primary"> Hello World! </p>

In the above example there are two attributes to the Paragraph tag , id is a identfier property for the tag and class is the attribute to add any style to the content.The value of the attribute can be anything which you want , the only rule is it cannot contain spaces . Also note that value is inside a "", you can put the value either inside a "" or '' both are valid way of defining the attribute.

You can have one or more attribute in a tag, also note the attributes are always added only to the opening tag and not to the closing tag.

Global Attributes

There are a set of attributes which can be used with all the tags , these attributes are called Global Attributes.You can refer to the link to see all the Global Attributes.

Below are some of the most commonly used attributes which we will go through.

  • id - this adds no visual change to the tag , it is used in javascript or css to identify the element we are referring to.
  • class- this is mostly used for styling the content in the tag, this is used for visual display of the content.
  • style- this is to add styling to the particular tag alone, we should try to avoid using this and instead use class.This can usefull for trying around different styles.
  • title-this provides tool tip or custom explanation for the tag.The value in the title is displayed when we do mouse over the content.

Lets do some coding

Open a text editor in your computer , it can be notepad if you are using Windows or Notes if you are using MAC.

Copy the below set of codes ,we will go through each of these in the next chapter.This is a baic skeleton for an HTML.

<!DOCTYPE html> <html lang="en"> <head> <title>My First Code</title> </head> <body> </body> </html>

Now we can add our code snippet <p> Hello World! </p> between the body tags.

<!DOCTYPE html> <html lang="en"> <head> <title>My First Code</title> </head> <body> <p> Hello World !</p> </body> </html>

Save the file as hello.html .Please note the extension , it should be .html file while saving .Once saved go to the file and right click and open it in any browser.

Voila!! your first HTML page is displayed Hello World!
